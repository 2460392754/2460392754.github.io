1.1 注册

使用class的static属性进行创建一个名为install的静态方法, 用于Vue.use注册插件。

插件注册时把当前环境的vue对象引用保存下来，确保项目和插件使用的是同一个引用对象。因为需要利用项目中的data功能来辅助state和getters值更新后触发视图更新。
